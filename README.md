# Geospatial Analysis of Health Equity in Massachusetts

## Project Overview
This project maps socio-economic and chronic disease correlations across Massachusetts using public datasets. By leveraging Python for data processing and Power BI for visualization, the analysis identifies and highlights healthcare service gaps for underrepresented populations through multi-variable geospatial tracking.

## Technical Stack
* **Languages & Processing:** Python, Pandas
* **Visualization:** Power BI (DAX)
* **Data Sources:** Massachusetts Public Health & Socio-economic Datasets

## Repository Contents
* `MA_Health_Equity_Analysis.ipynb`: The core Python notebook detailing the data cleaning, standardization, and exploratory data analysis (EDA) of the public health datasets.
* `MA_Health_Equity_PowerBI.pdf`: A static export of the interactive Power BI dashboard used to map vulnerabilities and visualize findings.
* `Geospatial Analysis of Health Equity in Massachusetts.pdf`: The comprehensive final report documenting the methodology, analytical findings, and actionable health policy insights.
* `requirements.txt`: Environment dependencies required to run the analytical notebook.

## Methodology & Results
1. **Data Engineering:** Ingested and cleaned massive public health datasets, handling missing values and normalizing socio-economic indicators for accurate comparison.
2. **Dashboard Development:** Engineered an interactive Power BI dashboard to overlay social vulnerability data with chronic disease prevalence.
3. **Actionable Insights:** Successfully surfaced critical healthcare service gaps, providing a data-driven framework for targeted community health interventions.
